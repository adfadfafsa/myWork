정의 목록

HTML
HyperText Markup Language 또는 HTML은 웹 브라우저에 표시되도록 설계된 문서의 표준 마크업 언어입니다. Cascading Style Sheets 및 JavaScript와 같은 스크립팅 언어와 같은 기술의 도움을 받을 수 있습니다.
CSS
종속형 시트 또는 캐스케이딩 스타일 시트는 마크업 언어가 실제 표시되는 방법을 기술하는 스타일 언어로, HTML과 XHTML에 주로 쓰이며, XML에서도 사용할 수 있다. W3C의 표준이고, 레이아웃과 스타일을 정의할 때의 자유도가 높다. 기본 파일명은 style.css이다.
